# مساهمة في Layer ∞

## 📋 قواعس السلوك

نلتزم بـ [Contributor Covenant](https://www.contributor-covenant.org/).

## 🔄 عملية المساهمة

### 1. Fork المشروع
```bash
git clone https://github.com/YOUR_USERNAME/layer-infinite-dollar1usd.git
cd layer-infinite-dollar1usd
```

### 2. إنشاء فرع للميزة
```bash
git checkout -b feature/your-feature-name
```

### 3. تثبيت المتطلبات
```bash
npm install
npm run build
```

### 4. إجراء التغييرات والاختبار
```bash
npm test
npm run lint
```

### 5. Commit بوضوح
```bash
git commit -m "feat: Add amazing feature"
git push origin feature/your-feature-name
```

### 6. فتح Pull Request

تأكد من أن PR يحتوي على:
- وصف واضح للتغييرات
- اختبارات جديدة
- تحديثات التوثيق
